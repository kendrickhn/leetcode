## Problem Summary
Given an array of integers and a number k, count how many pairs (i, j) exist such that:
- i < j
- nums[i] == nums[j]
- (i * j) is divisible by k

## Approach
Brute-force: Loop through all (i, j) pairs with i < j, check both conditions.

- Time: O(n^2)
- Space: O(1)
